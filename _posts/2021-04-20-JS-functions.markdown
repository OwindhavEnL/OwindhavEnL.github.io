- JavaScript 中，scope 是由 function 或是 code block 建立 
- 為了方法參數建立的 binding 或是在方法內宣告的 binging 只能在該方法內被參考，此稱為 local bindings
- 每個 function 呼叫會建一個新的 binding，呼叫同個方法不同次都會是一個新的 instance
